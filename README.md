# 就是Wan

就是玩APP首先是一个完整的实战项目，其中涵盖了各种Android开发技术的最佳实践（不限架构、业务功能、性能优化、黑科技等），项目基础工程采用组件化架构，在此基础上会以组件的形式逐渐实现各种技术的最佳实践。敬请期待💪💪💪
